# CleanVillageRecycling
Clean Village Recycling
Introduction
This project is an initiative dedicated to making our villages plastic-free and environmentally friendly. Our main objective is to create awareness among villagers and establish a structured method for collecting and recycling plastic waste.
Project Objectives
Ensure proper management of plastic waste in the village.
Simplify the waste collection process through the "Clean Village Plastic Bank".
Build a clean and eco-friendly village environment.
